# Our Story Images

This directory contains images for the "Our Story" page sticky scroll sections.

## Image Naming Convention

Please name your images as follows:
- `section-1.jpg` - How We Met
- `section-2.jpg` - Falling in Love
- `section-3.jpg` - Our First Date
- `section-4.jpg` - Adventures Together
- `section-5.jpg` - The Proposal

## Image Specifications

For best performance and quality:
- **Format:** JPG or WebP
- **Recommended Width:** 1200-1600px (images will scale to fit, preserving aspect ratio)
- **Aspect Ratio:** Any (native aspect ratio will be preserved - portrait orientation recommended)
- **File Size:** Keep under 500KB per image for optimal loading
- **Quality:** 80-85% JPEG quality is recommended

## Optimization Tips

Before adding images here:
1. Resize to recommended dimensions
2. Compress using tools like:
   - ImageOptim (Mac)
   - TinyPNG (Web)
   - Squoosh (Web - https://squoosh.app)

## Usage

These images are referenced in `src/components/AboutUs.vue` and will be served directly from the public folder at build time.
